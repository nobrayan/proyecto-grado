# Sistema de Concentración Solar para Secado de Cacao con Estacion Meteorológica

Este repositorio contiene el desarrollo de mi proyecto de grado, un sistema de **concentración solar** para el secado de cacao y otros granos. También incluye una **estación meteorológica** para monitorear las condiciones ambientales.  

Aquí encontrarás el código, esquemas y documentación del proyecto.
